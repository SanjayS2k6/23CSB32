# Notification System
## Project Overview
This is a simple Notification System made using HTML, CSS, and JavaScript.
It shows a notification bell with unread notifications. When the user clicks the bell, a notification panel opens. The user can read the notifications and mark them all as read.
## Features
* Notification bell icon
* Unread notification count
* Notification dropdown panel
* Mark all notifications as read
* Simple and responsive design
## Technologies Used
* HTML
* CSS
* JavaScript
